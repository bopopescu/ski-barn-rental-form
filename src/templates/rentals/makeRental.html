Please select a ski type based on the renters ability level.

Type 1

Type 1 riders generally prefer slower speeds on gentle slopes, due to the fact that they opt to set the binding so that it will release easier in slower speed falls. Oriented towards a beginner rider.

Type 2

Type 2 is the average for people who cannot identify as a type 1 or a type 3. Basically, type 2 is build to provide an average retention to release ratio for a variety of speeds and varied terrain.

Type 3

Type 3 riders are the agressive riders that would rather their binding to release less often to allow for less risk of inadvertent binding release in high speed/high stress situations.
